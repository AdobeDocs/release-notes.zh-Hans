## Experience Cloud

Experience Cloud 的新增功能和修复。

### Marketing Cloud ID 服务 {#mcvid}

本发行说明将 [!DNL  Marketing Cloud] ID 服务代码 (` VisitorAPI.js`) 增加为版本 2.0。该版本包含以下功能、修复和其他更改。

>[!IMPORTANT]
>
>默认情况下，ID 服务代码 v2.0 与 [!DNL  Adobe Advertising Cloud] 自动同步 ID。另请参阅[了解 ID 同步和匹配率](https://docs.adobe.com/content/help/en/id-service/using/home.html/mcvid-match-rates.html)。

修复：

* 修复了导致 AppMeasurement 无法对 Analytics 发起跟踪调用的错误。（MCID-254、MCID-256、MCID-286）
* 修复了一个在以下情况中 ID 服务无法立即失败的错误：访客启用了广告拦截器，并且将其配置为排除 [!DNL  demdex.net] 域。这种错误非常少见，因为大多数广告拦截工具并不拦截 [!DNL  demdex.net] 域。(MCID-233)
* 修复了由 ID 服务代码和客户网站上的自定义脚本之间的交互而导致的错误。此问题导致 Internet Explorer 9 无法加载网页。(MCID-206)

Marketing CloudID服务帮助位 [于此](https://docs.adobe.com/content/help/zh-Hans/id-service/using/home.html)。
