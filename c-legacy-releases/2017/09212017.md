**错误修复和其他更改**

* 修复了一个与 Chrome 相关的错误，在该浏览器中单击“返回”按钮时，该错误会导致 ID 服务出错。(CORE-5135)
* 现在，当事件调用响应中的区域 ID 发生更改时，ID 服务会重新触发 ID 同步。(CORE-4342)
* Added new documentation, [Content Security Policies and the Marketing Cloud ID Service](https://docs.adobe.com/content/help/zh-Hans/id-service/using/reference/csp.html) that explains how to allow calls to Adobe domains used by the ID service.

请参阅 [Marketing Cloud ID 服务](https://docs.adobe.com/content/help/zh-Hans/id-service/using/home.html)产品文档。

### Data Connectors {#section_190F5136147643AFB77E9CF1A067C438}

**ExactTarget 数据连接器与 Adobe Analytics 的集成**

Salesforce最近已停止ExactTarget Data Connector集成。 由于此决定，对于新的实现，Data Connectors接口中不再可访问ExactTarget向导。

ExactTarget集成对于之前配置了该集成的客户将继续可见。 现有实施将在未指定的时间长度内继续运行（由Salesforce确定）。 如果客户有问题，建议他们联系其Salesforce/ExactTarget代表。

**应对措施**

分析客户可以使用标准Analytics功能和API(包括分类、数据源、处理规则和Data warehouse导出)为ExactTarget实施自定义集成。 可以根据需要考虑与Adobe咨询团队的约定。
