# 发行说明 - 2016 年 3 月

>[!NOTE]

Experience Cloud 和核心服务的发行说明。

iOS SDK 版本 4.8.6（2016 年 3 月 9 日）包含以下更改：

| 功能 | 描述 |
|--- |--- |

有关产品文档，请参阅 [Adobe Mobile Services](https://docs.adobe.com/content/help/en/mobile-services/using/home.html)。要查看以前版本的发行说明，请展开左侧窗格中的“历史发行说明”**。

### Marketing Cloud ID 服务 {#section_5BA06A92D6E54715B025D86C192B2A8B}

2016 年 3 月 17 日

版本 1.5.4 包括以下更改：

| 功能 | 描述 |
|--- |--- |
| 选择退出支持 | Marketing Cloud ID 服务支持访客选择退出请求。(AAM-21725) |
| 更改 ID 同步时间间隔 | 现在，每当调用数据收集服务器时，Marketing Cloud ID 服务便会执行 ID 同步调用。以前，ID 服务仅在获得 Marketing Cloud ID 的首次调用中提出一次请求。(AAM-22934) |

**修复、增强功能和弃用功能**

* 修复了导致趋势报表显示当前日期前一天的结果的问题。此问题仅影响东欧时区。(AAM-22321)
* 修复了 Audience Marketplace 中阻止数据提供者激活私人无品牌数据馈送的错误。(AAM-23377)
* 将数据源创建工作流程中的 ID 类型选项（从“移动”）更改为“设备广告 ID”。(AAM-22947)
* 添加了 ` mathcesword`，这是一种新命名的运算符，有助于创建特质规则，而无需使用复杂的正则表达式。(AAM-22368)
