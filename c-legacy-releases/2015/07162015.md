# 发行说明 - 2015 年 7 月

了解 Adobe Marketing Cloud 的新增功能和修复。

## 核心服务

### Marketing Cloud 界面 {#marketing_cloud_interface}

Version 15.7 planned release: **July 22, 2015**

* 修复了导致无法在 Analytics 报表中更新“查看/编辑架构”页面（位于客户属性中）上指定的属性描述的问题。(MAC-25985)
* 修复了无法呈现已上传资产的缩略图的问题。(MAC-25863)
* 修复了导致在 Reports &amp; Analytics 中创建的新区段在 Marketing Cloud 受众中不可用的问题。 (MAC-25817)
* 修复了受众在使用访客 ID 服务时无法从 Analytics 共享的问题。（MAC-25788、MAC-25747）
* 在客户属性中添加了对多字节字符的支持。(MAC-25552)

**已知问题**

已知问题导致重复自动生成的帐户以Audience Manager方式创建，并自动将其关联到用户的Marketing Cloud标识。 如果您在关联帐户之前尝试导航到 Audience Manager，则会出现此问题。Adobe建议在导航到Audience Manager之前，将Marketing Cloud帐户链接到Audience Manager。 (MAC-25640)

请参阅 [Marketing Cloud 产品文档](https://docs.adobe.com/content/help/zh-Hans/core-services/interface/about-core-services/core-services-landing.html)，以获取产品帮助。

## Analytics

### Reports &amp; Analytics 修复 {#fixes_reports_analytics}


* 修复了在多个报表包中使用区段时发生的问题。 (AN-104828)
* 更改了在Admin Console中可显示的报表包数量限制，同时仍可显示自定义列（从500到400）。 (AN-102864)
* 修复了内容感知服务的问题 - 列大小已提升至 255，以便支持 UTF-8 编码数据。 (AN-103834)
* 修复了自定义转换报告菜单未在UI中加载的问题。(AN-103816)
* 修复了无法以 [!DNL  .pdf] 格式生成转化报表的问题。(AN-103037)
* 修复了某些报告中的图形显示未来一个月的数据的问题。 （AN-104372和AN-101257）
* 修复了显示区段不加载到仪表板报表中的问题。 (AN-105088)
* 修复了仪表板显示不可读行项的问题。 行项目在多字节字符的中间被截断。 (AN-104940)
* 修复了自定义日历无法返回正确数据的问题。 (AN-103825)
* 修复了PDF下载无法使用时间格式的指标的问题。 (AN-101515)
* 修复了导致包含区段的计算量度返回错误数据的问题。 (AN-105320)

### Report Builder 修复 {#fixes_arb}

* 修复了导致“总分钟数”计算量度未在用户界面中显示的问题。 (AN-105218)

### Ad Hoc Analysis 修复 {#fixes_ad_hoc}

* 修复了“城市”报告(GeoSegmentation)中缺少“城市”数据的问题。 (AN-104987)
* 修复了转化量度无法正确加载的问题。 (AN-104957)
* 修复了在创建计算度量时使用4个以上数学运算时出现的问题，该问题导致错误。 (AN-105362)

### 数据仓库修复 {#fixes_data_warehouse}

* 区段的固定变量／日期范围支持。 (AN-105271)
* 修复了未生成UTF-8字符的表格文件的问题。 (AN-75471)
* 修复了计数器事件度量重复消除每次点击具有多个产品的报表，以及产品分类细分。
* 来自数据仓库中页面查看次数量度的报表结果当前与 Reports &amp; Analytics 中的结果完全一致。(AN-97619)

### 点击流数据馈送修复 {#clickstream}

**2015 年 7 月 27 日更新**：我们正在更新数据馈送提交基础环境。我们正在向已用于传送数据服务的池中添加其他IP地址。 如果您的FTP站点上有数据源要传送到的IP白名单，您需要将这些附加IP地址添加到该白名单中。

* 伦敦：185.34.189.0/27，计划2015年8月13日增加)
* 奥克兰：192.243.248.32/27，计划于2015年8月20日添加
* 达拉斯：66.235.153.96/27，计划2015年8月27日增加

**修复、增强功能和弃用功能**

* 数据源API方法已迁移到Swagger。 See the [Audience Manager API Docs](https://bank.demdex.com/portal/swagger/index.html) for methods, syntax, and code samples.
* 修复了导致您无法正常地将特质从[!UICONTROL 受众市场]数据源添加到算法模型的问题 (AAM-16495)。
* 修复了在将区段映射到目标时，Firefox 和[!UICONTROL 目标生成器]中妨碍您添加开始日期的问题 (AAM-15785)。
* 修复了区段对话框中显示错误区段填充号码的问题 (AAM-15809)。
* 修复了在拖放编辑器和代码视图之间进行切换时，[!UICONTROL 特质生成器]中错误地显示复杂特质规则的问题 (AAM-15919)。

* 改进了基本信息窗口中目标的标签。标签由[!UICONTROL 限制]更改为[!UICONTROL 数据导出标签] (AAM-19824)。

**修复、增强功能和弃用功能**

* 修复了有时会导致“[!UICONTROL 帖子分析]”报表中的下载和发送功能失效的问题。
* 解决了许多本地化问题，从而提高了用户界面内文本的语言质量，其中包括对术语的标准化，改进翻译以使文本更好地符合上下文，为翻译添加更多的上下文，修改语法和标点错误，以及将日期/时间字符串本地化。
