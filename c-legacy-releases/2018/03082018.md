### [!DNL Experience Cloud] ID 服务 {#mcvid}

Marketing Cloud ID 服务的改进和修复。

* 修复了函数在子 *`resetBeforeVersion`* 域上不起作用的错误。 (CORE-11390)
* 修复了以下错误：当 [!DNL  visitor.cookieDomain] 具有前导点时，Experience Cloud ID 服务会错误地在不包含前导点的域上设置 AMCV Cookie。例如，除了设置AMCV [cookie外](https://marketing.adobe.com/resources/help/en_US/whitepapers/cookies/cookies_mc.html) , [!DNL  sampledomain.com] 还在上设置 [!DNL  .sampledomain.com]。 (CORE-10634)

请参阅 [Experience Cloud ID 服务](https://docs.adobe.com/content/help/zh-Hans/id-service/using/home.html)产品文档。

### [!UICONTROL Mobile] Services {#mobile}

[!UICONTROL Mobile] Services 的新增功能、更新和修复。
