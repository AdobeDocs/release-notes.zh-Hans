### [!UICONTROL 动态标签管理]

[!DNL Adobe] 计划在 2020 年底停用[!UICONTROL 动态标签管理]。

* **2019-07-01：** 从 2020 年 7 月起，DTM 将不再允许创建新属性。这不会对现有属性产生任何影响。
* **2020-07-01：** DTM 属性将进入只读模式。您将无法创建或编辑工具、规则或数据元素。您将无法再将库发布到 DTM 环境。这不会对之前已发布的库产生任何影响。

有关更多信息，请参阅 [DTM 停止支持计划](https://forums.adobe.com/community/experience-cloud/platform/launch/blog/2018/10/05/dtm-plans-for-a-sunset)。

我们已将“出站历史记录报表”的名称更新为“出站文件历史记录报表”。先前的名称会给一些客户带来误导，让他们误认为该报告显示的是 HTTP 目标的出站数据。事实上，该报告涵盖了交付到 S3 或 FTP 位置的文件。

**已知问题**

* Safari 的最新版本包括了“智能防跟踪功能”(ITP) 2.0 工具。该工具会对 Safari 用户的“可寻址受众”量度以及通过 h_referer 信号进行的数据收集产生影响。有关 Safari 流量的信息，请参阅[可寻址受众匹配率较低的原因](https://marketing.adobe.com/resources/help/en_US/aam/addressable-audiences-match-rates.html)和[使用 h_ 前缀的数据收集](https://marketing.adobe.com/resources/help/en_US/aam/r_tb_variable_prefixes.html)。
* [算法建模中的特征排除](https://marketing.adobe.com/resources/help/en_US/aam/trait-exclusion-algo-models.html)的发布，给使用[基于角色的访问控制](https://marketing.adobe.com/resources/help/en_US/aam/c_administration.html)的客户带来了问题。当您在新建模型时，如果只选择您有权访问的数据源，那么可以在 **排除** 窗口中看到与之对应的特征。然而，假如除了选择您具备访问权限的数据源之外，还选择了不具备访问权限的其他数据源，那么您看到的将会是一个空列表。为了查看特征，请您取消选择不具备访问权限的数据源。(AAM-42380)
