# 发行说明 - 2018 年 1 月

Adobe Experience Cloud 的新增功能和修复。

>[!NOTE]
>
>若要提前收到发行说明的通知，请订阅 [Adobe 产品更新早知道](https://www.adobe.com/subscription/priority-product-update.html)。优先级产品更新在发布前3到5个工作日发送。 请在发布时检查更新。 对于在版本发行之后发布的新信息，将使用发布日期进行标记。

## Experience Cloud 脚本

| 用例 | 发布日期 | 描述 |
