### [!DNL Experience Manager Sites] as a [!DNL Cloud Service]

_新增功能_

* 将内容片段作为 JSON 产品建议从 AEM as a Cloud Service 导出到 Adobe Target。<!-- OLD. UPDATE ON JUNE 7 -->
* 现在，利用对 GraphQL 分页和排序的支持以及内部缓存增强功能，可以在使用复杂的 GraphQL 查询和筛选条件从 AEM 获取大型内容集时帮助提高已解耦客户端应用程序的性能。<!-- OLD. UPDATE ON JUNE 7 -->

### [!DNL Experience Manager Assets] as a [!DNL Cloud Service]

_新增功能_

* 推出了面向 Dynamic Media 视频交付（已启用 CMAF）中的自适应流式处理的新协议（DASH – 基于 HTTP 的动态自适应流式处理）支持：<!-- OLD. UPDATE FOR JUNE 7 -->
   * 自适应流式处理 (DASH/HLS) 可确保最终用户获得更出色的视频观看体验。<!-- OLD. UPDATE FOR JUNE 7 -->
   * DASH 是自适应视频流式处理的国际标准协议，在业界得到广泛应用。<!-- OLD. UPDATE FOR JUNE 7 -->
   * 在亚太地区和北美可用；[通过提交支持工单的方式启用](https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/assets/dynamicmedia/video.html#enable-dash)。即将在欧洲、中东和非洲推出。<!-- OLD. UPDATE FOR JUNE 7 -->
* 添加了对 WebP 图像的支持以自动提取元数据、生成缩略图和自定义演绎版。现在也对这些文件支持智能标记和智能裁切功能。<!-- OLD. UPDATE FOR JUNE 7 -->
