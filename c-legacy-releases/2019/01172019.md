### [!UICONTROL Analysis Workspace] {#aa-wksp}

[!UICONTROL Analysis Workspace] 的功能和修复。

#### [!UICONTROL 同类群组分析]

对同期群分析 [!UICONTROL 的重大改] 进使您能够：

* 分别应用细分和返回指标。
* 展示客户流失而非保留。
* 显示延迟表(包含事件前后经过的时间)。
* 自定义同期群维度(根据eVar对访客进行分组，而不仅仅是时间)。
* 滚动同期群计算： 根据上一时间段（而非原始同期群）计算保留期和客户流失率。
* 在包含和返回字段中添加多个指标，并应用区段。 （不支持计算指标。）

#### [!UICONTROL 视图密度]

此新设置可减少左边栏、自由格式表和队列表中的垂直边距，从而使您能够在单个屏幕上查看更多数据。

This setting is accessible via **[!UICONTROL Project]** > **[!UICONTROL Project Info &amp; Settings]**.

#### 在归因 IQ 中支持多值变量

Analytics 中的某些维度可以在单次点击中包含多个值，例如 listVar、产品变量、列表属性或推销 eVar。Analysis Workspace 允许您在点击级别将归因 IQ 应用于任何此类变量。

#### [!UICONTROL 营销渠道]增强功能

现在，我们向营销渠道处理规则显示 AMO ID 和 AMO EF ID。这些是 [!DNL Advertising Cloud] 和 [!UICONTROL Advertising Analytics] 集成使用的主要和次要跟踪代码。

#### 性能改进

加快细分可视化的改进。 例如，具有多个细分的项目加载速度更快。

#### [!UICONTROL Analysis Workspace] 的修复

* 修复了通过将组件拖放到面板拖放区域中创建的内部区段定义不会跨会话保留的问题。 (AN-172110)

## [!DNL Advertising Cloud] {#adcloud}

[!DNL Adobe Advertising Cloud] 的新增功能。

### 项目组合

（测试版）新的“每周”支出策略可让您在尝试支出指定周数时最大化加权收入。 此策略可自动使用“每周时间”模型和模拟。

如果要对一个作品集使用此策略，请等到该作品集优化至少两周，然后主动监控支出和性能，直到该功能被验证。

您现在可以允许Google Ads使用新的活动支出策略“GoogleTargetCPA”和“GoogleTargetROAS”优化多个目标，以实现单次广告支出回报(ROAS)或每次收购成本(CPA)。 通过这些支出策略，所有分配给Google Ads活动的都将具有指定的竞价策略和指定的目标; 其他搜索引擎上的活动被忽略。 所有其他包设置均被禁用。

在“自动优化竞价调整值”组合部分中，“受众Target”选项现在在活动和广告组级别优化Google Ads再营销列表、客户匹配和类似受众（您可以在Google Ads中创建）。 以前，“再营销列表”选项仅优化了广告组级别的再营销列表。

对于启用了学习功能的包，Advertising Cloud现在可以在指定的学习预算内自动竞价报价单位，且零印象。

默认情况下，您可以使用旧版手动规则竞价零展示次数的投标单位，直到弃用手动选项。 在这种情况下，学习预算仅适用于具有模型的投标单位，而零印象学习使用额外支出。 该项目组合可能花费超过分配的学习预算，以满足指定的参数。

### 搜索促销活动

活动（旧版）视图不再位于主菜单中。 要使用旧版视图（将在2月份弃用），请转到“搜索”>“活动”>“活动”，滚动到页面右下方，然后单击“旧版活动”。

如果您在旧视图中，则可以从主菜单打开“搜索”>“活动”>“活动”，或滚动到右侧并单击“单击此处”尝试新的活动体验，以返回到新视图。

（仅限Yandex帐户）当帐户由代理／管理帐户管理时，您现在可以使用“MCC帐户”字段指定管理帐户。 要删除现有关联，请选择“无MCC帐户”。

(Google Ads)现在，您可以为搜索和显示网络上的活动创建响应式搜索广告（在Google Ads中处于测试模式）。 Google Ads动态组合一组广告标题和描述中基于文本的响应式搜索广告，有利于组合表现良好。 您可以选择将广告标题和说明固定到特定位置。 报告处于广告级别。

关于针对管理响应式搜索广告的批量工作表支持，将在未来版本中提供。有关将促销活动分配到组合的数据现在包含在组合更改历史记录中，该历史记录可从“组合”视图中获取。在比较两个日期范围的数据时，您现在可以按比较列（如展示次数R2）对数据进行排序。

错误消息现在可以选择报告问题。 会话数据会自动包含在报告中，您可以选择输入文本详细信息。

### 搜索约束测试版

(仅限Google Ads和Bing Ads账户； 测试版功能)现在，您可以根据投标者的印象份额来限制投标。

### 社交测试版

您现在可以使用工具栏中的编辑按钮编辑多个广告集的预算和状态。

### 报表

为了支持Google Ads扩展文本广告中可选的第三个标题，广告变化报告现在包括“Creative Title3”和“Description2”字段。

(针对Google Ads和Bing Ads活动的开放测试版)两份新的专业报告提供印象共享指标： 关键字每日印象共享报告和活动每日印象共享报告。 Advertising Cloud正在寻求客户对报告的反馈。
