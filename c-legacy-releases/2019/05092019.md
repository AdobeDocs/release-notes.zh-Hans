### Adobe Analytics 的新增功能和修复 {#aa-features}

| 功能 | 描述 |
| -----------| ---------- |  
| **AppMeasurement 版本 2.14.0** <ul><li>修复了当多个命中等待处理时，管理跟踪器参数状态的问题。(AN-176931、AN-176629、DTM-12758)</li><li>更新了 AppMeasurement 以包含 Visitor.js 4.3.0 (AN-180049)</li></ul> |
| **Analysis Workspace：**&#x200B;新增&#x200B;_包含重复实例_&#x200B;流量可视化图表设置 | “包 _括重复实例_ ”(Include Repeat Instances)流设置允许您选择包括或排除重复实例，如“页面重新加载”(Page Reloads)。 此外，所有流可视化现在仅基于实例。 |
| **Ad Hoc Analysis：**&#x200B;与 Java 11 兼容 | 现在，Ad Hoc Analysis 已与 Java 11 兼容。了解如何在Java [11上运行Ad Hoc Analysis](https://marketing.adobe.com/resources/help/en_US/dsc/adhoc-java.html)。 |
| **数据收集：**&#x200B;新增的 s_ecid cookie | Added a new [first-party server cookie](https://docs.adobe.com/content/help/zh-Hans/core-services/interface/ec-cookies/cookies-analytics.html), s_ecid, in which data collection stores the visitor&#39;s ECID. |

**Analysis Workspace 修复**

* 修复了影响网页逗留时间的问题。在计算“所用时段”时，工作区报表将不再使用页面名称，从而可以计算粒度和分时段的点击。 (AN-140479)
* 修复了线路可视化性能问题，这是提高工作区性能的更大努力的一部分。 (AN-174878)
* 修复了下载的 .csv 文件中缺少 UTF-8 编码的问题。(AN-178393)
* 修复了Workspace项目性能较慢的问题。 (AN-177710)
* 修复了Y轴粒度范围较小的线可视化显示问题。 (AN-176467)

**其他 Analytics 修复**

* Audience Analytics:修复了在Audience Manager(AAM)中更改受众名称后发生的问题——更新的名称未反映在Audience Analytics中。 (AN-176237)
* 修复了阻止用户在AAM中保存Analytics区段的问题。 这是由现有的AAM文件夹混合大小写名称造成的。 现在，我们将所有文件夹都视为不区分大小写，以便它们能够同步。 (AN-177934)
* 修复了用户通过Experience Cloud登录Analytics后会话超时时发生的问题。 恢复会话时，用户被重定向到错误的URL。 (AN-176812)
* 修复了Data warehouse请求中时区偏移的问题。 (AN-177585)
